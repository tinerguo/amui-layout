# gulptemplate

> gulp + webpack 多页面项目

### 安装淘宝的cnpm 命令
```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```
### 安装全局的gulp 命令
```
cnpm i gulp -g
```


## 构建安装
> 在项目文件夹目录下执行
```



# 安装依赖
cnpm install

# 运行项目 访问路径：http://localhost:9000或者http://[本机IP]:9000
gulp serve

# 项目打包
npm run lib

# 项目发布
npm run push
```
